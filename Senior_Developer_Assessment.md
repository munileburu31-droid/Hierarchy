# Senior Developer Take-Home Assessment (Java)

**Time limit**: 1 day from receipt

**Java Version**: 21

## Tasks

Both tasks are required.

### Task 1: Hierarchy Filter Implementation

**File**: `Hierarchy.java`

Implement the `filter()` method in the `HierarchyFilter` class for the Hierarchy data structure and add comprehensive test cases. See inline comments in the file for detailed requirements.

- Use JUnit 5 or any testing framework you prefer
- Document any assumptions you make

### Task 2: Code Review

**File**: `SimpleCache.md`

Review the SimpleCache implementation provided in the file. Identify production issues for a high-concurrency environment (thousands of reads/sec, hundreds of writes/sec, tens of concurrent threads).

- Provide your review as a numbered list in `SimpleCache.md`
- Explain each issue and its potential impact

## Submission

1. Create a **public** GitHub repository with your solution
2. Structure the repository however you prefer
3. Provide the repository link

## Evaluation Criteria

Your submission will be evaluated on:
- Correctness
- Code quality
- Test coverage and edge cases
- Knowledge of Java and concurrency
